javaScript introduction

  1. 起源：为了将网页中的验证功能从服务器端转变到前端(browser)，由于以前网速慢。javaScript由网景公司发明，起初名为liveScript，后SUN公司介入更名为JavaScript。
  2. 96年微软实现JScript。
  3. 为了确保不同浏览器上运行的JavaScript标准一致，所以几个公司共同制定了JS的标准并命名为ECMAScript(简称ES标准)。
  4. 不同浏览器厂商对该标准会有不同的实现方式(不同的引擎)。
  5. 一个完整的JavaScript实现应由3部分组成：
    1)、ECMAScript标准+
    2)、DOM文档对象模型+
    3)、BOM浏览器对象模型。
  6. JS特点：
    1）、解释型语言；直接运行，不用编译；
    2）、动态语言；
    2）、基于原型的面向对象；
